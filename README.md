# LeetCode Solutions

A personal repository for tracking LeetCode solutions, notes, patterns, and progress.

## Structure

```text
.
├── solutions/
│   ├── python/
│   └── javascript/
├── templates/
└── notes/
```

## Naming Convention

Use this format for solution files:

```text
0001-two-sum.py
0001-two-sum.js
```

Recommended file header:

```text
Problem:
Link:
Difficulty:
Topics:
Time:
Space:
```

## Workflow

1. Add the solution under the relevant language folder.
2. Update the progress table.
3. Add notes for patterns, mistakes, or alternate approaches.
4. Commit with a clear message.

Example commit:

```bash
git add .
git commit -m "Add solution for Two Sum"
```
