# Patterns

Use this file to summarize recurring solution patterns, tradeoffs, and mistakes.

## Hash Map Lookup

- Useful when searching for complements, prior state, or frequencies.
- Common problems: Two Sum, Valid Anagram, Subarray Sum Equals K.

## Two Pointers

- Useful on sorted arrays, linked lists, and paired scans.
- Common problems: Valid Palindrome, 3Sum, Container With Most Water.

## Sliding Window

- Useful for contiguous subarray or substring constraints.
- Common problems: Longest Substring Without Repeating Characters, Minimum Window Substring.
